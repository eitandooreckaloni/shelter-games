# 🏠 Shelter Games

Web mini-games for families during shelter time. Safe, stress-reducing activities with zero-friction access.

## 🎮 Games Available

- **🧠 Memory Match** - Classic card matching game for memory training
- **📚 Word Play** - Simple word puzzles and vocabulary games  
- **🎨 Draw & Color** - Digital drawing canvas and coloring templates
- **🌸 Breathing Buddy** - Guided breathing exercises for relaxation
- **🎵 Rhythm Tap** - Musical rhythm games and sound activities
- **📖 Story Time** - Interactive choose-your-own-adventure stories

## ✨ Features

- **Zero-friction access** - No signup or registration required
- **Offline-ready** - Works with poor or no internet connection
- **Child-safe** - Family-friendly content designed for all ages
- **Stress-reducing** - Activities specifically chosen for calming effect
- **Responsive design** - Works on all devices (phones, tablets, laptops)
- **Shareable links** - Each game has its own URL for easy sharing

## 🏗️ Tech Stack

- **Next.js 16** - React framework with app router
- **Tailwind CSS** - Utility-first CSS framework
- **TypeScript** - Type-safe JavaScript
- **Vercel** - Deployment platform

## 📝 Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
